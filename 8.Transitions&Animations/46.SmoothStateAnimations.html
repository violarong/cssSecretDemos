46.Smooth state animations

主要知识点：animation-play-state
常见做法
@keyframes panoramic { to { background-position: 100% 0; } } .panoramic { width: 300px; height: 300px; background: url("images/bg-dq.png"); background-size: auto 100%; animation: panoramic 10s linear infinite alternate; animation-play-state: paused; } .panoramic:hover, .panoramic:focus { animation-play-state: running; }
