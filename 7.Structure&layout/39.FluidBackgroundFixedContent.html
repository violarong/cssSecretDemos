39.Fluid background, fixed content

主要知识点：:first-child:nth-last-child
常见做法
这是内容
.m-footer{background-color:#45b095;height:300px;} .m-footer .wrapper{ max-width:300px; margin:1em auto; height:100%; } .m-footer .wrapper .content{ background: #9e9cf6; height: 100%; color: white; }
只使用一个html标签的做法
这是内容
.m-footer1{ padding: 0 1em; padding: 0 calc(50% - 150px); background: #45b095; height:300px; } .m-footer1 .content{ background: #9e9cf6; height: 100%; color: white; }
